Retail Sales Performance Dashboard

Project Overview
This project demonstrates an end-to-end business intelligence solution developed in Microsoft Excel.
The dashboard transforms raw retail sales data into an interactive executive report using Power Pivot, Pivot Tables, Pivot Charts and Slicers.

Tools Used
* Microsoft Excel
* Power Pivot
* Pivot Tables
* Pivot Charts
* Slicers
* Data Modelling
* Data Cleaning

Dashboard Features
* Executive KPI Cards
* Revenue Analysis
* Profit Analysis
* Monthly Trend Analysis
* Interactive Channel Filter
* Interactive Month Filter
